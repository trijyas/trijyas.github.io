Grid App for Artists

Grid App for artist helps you to create detailed artworks by providing proportion idea through grid over your reference images.

How to Use ?

Setup Canvas

1. Select canvas size > crop image > add border if required

2. Apply filters or adjust image properties

Adjust Grid

1. Set grid box size

2. Adjust style, size/thickness, colour and opacity of line, diagonal and label

Tip: You can adjust grid property different for all three.

All Set

You're all set! Enjoy drawing with help of Zoom, Lock and Real Size.
